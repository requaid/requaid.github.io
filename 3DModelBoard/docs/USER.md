# 사용 매뉴얼 — 단순 리더(구경하는 분)

이 게시판은 단순히 구경만 해도 되도록 만들어졌습니다. 별도 설치나 계정, 토큰 없이 바로 사용할 수 있습니다.

---

## 1. 접속

- 공개 주소 예시: `https://<username>.github.io/3DModelBoard/`
- 최신 Chrome / Edge / Firefox / Safari에서 동작합니다.
- 모바일에서도 동작하지만 3D 모델 렌더링은 데스크톱에서 더 쾌적합니다.

## 2. 목록 페이지 둘러보기

![flow](https://img.shields.io/badge/flow-list-blue)

- 카드 하나 = 게시물 하나. 썸네일, 제목, 포맷 배지(**VRM / PMX / PMD**), 작성자, 날짜가 보입니다.
- 검색창에 키워드를 입력하면 **제목·작성자·태그**에서 즉시 필터됩니다.
- 상단 포맷 칩(**전체 / VRM / PMX / PMD**) 으로 좁혀볼 수 있습니다.
- 카드에 **[로컬]** 배지가 붙어 있으면 *당신의 브라우저에만* 저장된 게시물입니다. 다른 사람에게는 보이지 않습니다.
- 카드를 클릭하면 상세 뷰어로 이동합니다.

## 3. 뷰어 페이지 다루기

카드를 누르면 열리는 전체 화면 뷰어입니다.

### 카메라 조작
| 동작 | 마우스 | 터치 |
| --- | --- | --- |
| 회전 | 좌클릭 드래그 | 한 손가락 드래그 |
| 확대/축소 | 휠 | 두 손가락 핀치 |
| 패닝 | 우클릭 드래그 | 두 손가락 드래그 |

### 내장 모션
하단 바에서 버튼 하나만 누르면 됩니다.

- 🧍 **스탠딩** — 가볍게 숨 쉬는 정지 포즈
- 🚶 **걷기** — 일정 속도로 걷기
- 🏃 **달리기** — 빠른 러닝
- 💃 **춤추기** — 4초 길이 루프 댄스
- ⏹ **정지** — 현재 모션 중단 + T-포즈 복귀

**재생 중인 같은 버튼을 다시 누르면 정지**됩니다.

> 일부 모션은 모델에 필요한 본(뼈대)이 없으면 상태 표시에 "본을 찾을 수 없음"이라 나오고 적용되지 않습니다. 다른 버튼을 눌러 보세요.

### 모션 파일 드래그&드롭

`.vmd` 또는 `.bvh` 파일을 뷰어 캔버스 위에 **드래그해서 떨어뜨리면** 모델이 그 동작을 재생합니다.

- 파일은 **서버로 전송되지 않고** 브라우저 안에서만 처리됩니다.
- 최대 크기: 10MB
- 올바른 파일이 아니면 경고 토스트가 뜨고 아무 변화가 없습니다.
- 다른 모션(내장 포함)을 재생하면 드롭 모션은 자연스럽게 대체됩니다.

### 우상단 아이콘

- 📷 **스냅샷 저장** — 지금 보이는 화면을 PNG 파일로 다운로드합니다. (로컬 다운로드일 뿐, 어디에도 업로드되지 않음)
- ↩ **목록으로** — 목록 페이지로 돌아갑니다.

## 4. 자주 겪는 문제

| 증상 | 해결 |
| --- | --- |
| 화면이 검정색에 모델이 안 보여요 | 페이지가 아직 로딩 중일 수 있습니다. 10초 정도 기다려 주세요. 그래도 그러면 새로고침. |
| 모션 버튼을 눌러도 움직이지 않아요 | 모델이 해당 본을 갖고 있지 않은 것입니다. 다른 버튼을 눌러 보세요. |
| VMD를 드롭했는데 "본 매핑 불일치" | 해당 VMD가 비표준 본을 사용합니다. 다른 VMD로 시도해 주세요. |
| 모바일에서 느려요 | VRM 파일이 큰 경우 발열·성능 한계입니다. 데스크톱에서 열어 보세요. |

## 5. 개인 용도로 이 뷰어를 쓰고 싶다면?

게시 없이도, 당신 기기에만 저장해 개인적으로 모델을 돌려볼 수 있습니다.

1. 상단 **[+ 새 모델 올리기]** 또는 **글쓰기** 메뉴로 이동.
2. 모델 파일을 드롭/선택해 미리보기.
3. **💾 내 브라우저에 저장** 버튼만 누릅니다 *(PR로 공유하기는 누르지 마세요)*.
4. 목록으로 돌아가면 **[로컬]** 배지가 붙은 카드가 보입니다.
5. 이 카드는 *당신의 브라우저*에서만 보이고, 다른 사람에겐 보이지 않으며, 쿠키를 지우면 사라집니다.

> 더 깊은 기능(PR로 공유, 설정, 보안)은 [작성·운영자 매뉴얼](AUTHOR.md)을 참고하세요.
