# 완전 초보자용 사용 가이드 — 구경만 하는 분

이 문서는 **컴퓨터 프로그래밍을 전혀 몰라도** 이 게시판을 편하게 둘러볼 수 있도록 만든 안내서입니다.
어려운 용어가 나오면 그 자리에 쉬운 말 풀이를 붙였고, 끝에도 [용어사전](#9-용어-사전)을 따로 두었습니다.

게시글을 *직접 올리고 싶다면* 이 문서가 아니라 **[초보자용 작성자 가이드 (BEGINNER-AUTHOR.md)](BEGINNER-AUTHOR.md)** 를 보세요.

---

## 1. 이 사이트가 뭔가요?

인터넷에 올라와 있는 사람 모양 3D 캐릭터(주로 VRM, PMX, PMD 파일)를 **브라우저 안에서 바로 돌려 보고 걷게 하거나 춤추게 할 수 있는** 게시판입니다.

- **다운로드·설치 필요 없음** — 웹 주소를 여는 것만으로 끝입니다.
- **회원가입·로그인 불필요** — 그냥 보기만 한다면 아무것도 입력하지 않아도 됩니다.
- **서버 없음** — 당신이 여는 건 "정적 웹페이지"뿐입니다. 모델이 움직이는 계산은 전부 *당신 컴퓨터의 브라우저 안에서* 일어납니다.

> 💡 **정적 웹페이지란?** 은행 사이트처럼 서버가 매번 계산해서 보여주는 게 아니라, 미리 만들어 둔 HTML 파일을 그대로 열어 보는 것을 말합니다. 네트워크 부하가 거의 없고, 당신의 조작이 외부로 전송되지 않습니다.

---

## 2. 접속하기

운영자가 알려준 주소(예시: `https://어떤계정이름.github.io/3DModelBoard/`)를 크롬/엣지/파이어폭스/사파리 같은 **최신 브라우저**에 붙여넣고 엔터를 누르세요.

**권장 환경**

| 환경 | 사용 가능 여부 |
| --- | --- |
| 윈도우 PC · 맥 (크롬/엣지/파이어폭스) | ✅ 가장 쾌적 |
| 사파리 (맥) | ✅ |
| 아이폰·아이패드 (iOS 사파리) | ✅ 작동은 하지만 무거울 수 있음 |
| 안드로이드 폰 (크롬) | ✅ 작동은 하지만 무거울 수 있음 |
| 인터넷 익스플로러 | ❌ 지원하지 않음 |

사이트가 열리면 카드가 나란히 늘어선 **목록 페이지** 가 먼저 보입니다.

## 3. 목록 페이지에서 둘러보기

### 3-1. 화면 구성

![개념 그림: 상단에 '+ 새 모델 올리기', 아래에 카드들이 격자로 정렬]()

- 상단 바 — 사이트 이름, **[+ 새 모델 올리기]** 버튼, 메뉴 링크.
- 검색창 — 제목·작성자·태그를 입력하면 즉시 해당 카드만 남습니다.
- 포맷 칩 — **전체 / VRM / PMX / PMD** 중 하나를 눌러 특정 포맷만 보기.
- 카드 — 썸네일 + 제목 + 포맷 배지 + 작성자 + 날짜.

### 3-2. 카드에 붙는 표시

- **[VRM]·[PMX]·[PMD]** — 모델 파일 종류입니다. 당신이 봐야 할 차이는 없고, 참고용 색상 배지일 뿐.
- **[로컬]** — *당신의 브라우저에만* 저장된 게시물입니다. 다른 사람 눈에는 보이지 않고, 쿠키/데이터를 삭제하면 함께 사라집니다. 개인 보관용으로 썼다는 뜻입니다.
- **#태그** — 작성자가 달아둔 태그 중 2개까지 카드에 표시됩니다.

카드를 클릭하면 해당 모델의 **뷰어(viewer) 페이지** 로 이동합니다.

## 4. 뷰어 페이지 — 모델 돌려 보기

### 4-1. 카메라 움직이기

| 하고 싶은 동작 | 마우스로 | 화면을 터치로 |
| --- | --- | --- |
| 모델을 빙 돌려 보기 | **왼쪽 버튼 누르고 드래그** | 한 손가락으로 드래그 |
| 가까이/멀리 보기 | **마우스 휠 굴리기** | 두 손가락 **핀치(벌리기·오므리기)** |
| 모델을 옆/위로 살짝 옮기기 | **오른쪽 버튼 누르고 드래그** | 두 손가락으로 함께 드래그 |

> 💡 **팁**: 작게 보이면 휠을 앞쪽으로 굴려 확대하세요. 너무 가까이 들어가면 모델이 잘립니다. 그럴 때는 반대로 뒤로 굴립니다.

### 4-2. 배경이 보이는 게시물

작성자가 배경을 설정한 게시물이라면, 모델 뒤로 이미지가 깔려 보입니다. 배경은 **읽기 전용**으로, 당신이 바꾸거나 숨길 수는 없습니다.

> 외부 웹 주소를 배경으로 쓴 게시물은 해당 주소의 서버로 이미지 요청을 한 번 보냅니다. 프라이버시가 걱정되면 네트워크를 끊은 상태에서 여세요(그러면 배경만 비워진 채 모델은 잘 보입니다).

### 4-3. 움직이게 하기 — 하단 버튼

화면 아래쪽에 이런 버튼이 있습니다.

- 🧍 **스탠딩** — 가볍게 숨 쉬는 정지 자세
- 🚶 **걷기** — 제자리에서 걷는 동작
- 🏃 **달리기** — 제자리에서 뛰는 동작
- 💃 **춤추기** — 4초짜리 루프 댄스
- ⏹ **정지** — 움직임을 멈추고 T-포즈(팔 벌린 자세)로 돌아감

**같은 버튼을 한 번 더 누르면 재생이 멈춥니다.**

> 버튼을 눌렀는데 아무 일도 안 일어나고 상태 표시에 "본을 찾을 수 없음"이 뜨면, 그 모델이 해당 동작에 필요한 뼈대를 갖고 있지 않은 경우입니다. 다른 버튼을 눌러 보세요.

### 4-4. 내 모션 파일 넣어보기 (드래그&드롭)

윈도우 탐색기/맥 파인더에서 `.vmd` 또는 `.bvh` 파일을 **뷰어 화면 위에 그대로 끌어다 놓으면** 그 동작이 재생됩니다.

- 최대 10MB 까지.
- 파일은 **당신 컴퓨터 밖으로 나가지 않습니다.** 브라우저 안에서만 처리됩니다.
- 형식이 맞지 않는 파일은 경고 메시지가 뜨고 아무 변화가 없습니다.

### 4-5. 스크린샷 저장하기

화면 오른쪽 위 **📷** 버튼을 누르면 지금 보이는 장면이 PNG 이미지로 다운로드됩니다. 어디에도 업로드되지 않고, 당신 컴퓨터 "다운로드" 폴더에 바로 저장됩니다.

### 4-6. 목록으로 돌아가기

화면 오른쪽 위 **↩** 버튼 또는 브라우저 뒤로가기 버튼.

## 5. 내 컴퓨터에만 저장해서 혼자 즐기기

인터넷에 올리지 않고 **당신 기기에만 저장해서 개인용 뷰어로 써도 됩니다.** 이 경우 PAT(뒤에서 설명)도 깃허브 계정도 필요 없습니다.

### 5-1. 순서

1. 상단 **[+ 새 모델 올리기]** 버튼 클릭 → 글쓰기 페이지로 이동.
2. 모델 파일을 화면 위에 끌어다 놓거나(드롭) 클릭해서 선택.
3. 오른쪽 미리보기에 모델이 나오면 OK.
4. 제목만 써도 됩니다(작성자·태그·설명은 생략 가능).
5. 원하면 **🖼 이미지 업로드** 로 썸네일을 직접 지정, **🏞 배경 이미지 선택** 으로 배경 이미지를 지정할 수도 있습니다. 안 해도 됩니다.
6. **💾 내 브라우저에 저장** 버튼을 누릅니다. (아래 **🚀 PR로 공유하기** 는 누르지 마세요 — 이건 전세계 공개용입니다.)

저장 완료 토스트가 뜨고 목록으로 돌아오면 **[로컬]** 배지가 붙은 카드가 새로 생겨 있습니다.

### 5-2. 로컬 저장의 주의점

- 당신의 **브라우저 한 곳에만** 있습니다. 다른 기기/다른 브라우저에서는 안 보입니다.
- **브라우저 데이터를 지우면 같이 사라집니다** — 시크릿 모드, "쿠키와 사이트 데이터 삭제", 다른 사용자 프로필에서는 보이지 않습니다.
- 카드에 마우스를 올리면 오른쪽 위에 **삭제** 버튼이 나타납니다. 이건 *내 브라우저에서만* 지우는 것이지 다른 사람에게는 영향이 없습니다.

### 5-3. 개인 보관 한도

- 모델 파일 크기: 최대 100MB
- 배경 이미지: 최대 5MB
- 썸네일 이미지: 최대 2MB
- 모션 파일: 최대 10MB

브라우저마다 총 저장 한도가 다르고(보통 수 GB), 한도에 가까워지면 브라우저가 경고를 띄우기도 합니다.

## 6. 자주 있는 문제와 해결법

| 증상 | 원인 / 해결 |
| --- | --- |
| 페이지가 하얀 채로 안 뜸 | 브라우저가 너무 구형. 최신 크롬/엣지/파이어폭스로 재접속. |
| 카드는 보이는데 모델이 로딩 중에서 멈춤 | 최대 45초까지 기다린 뒤 자동으로 **에러 팝업**이 뜹니다. 팝업이 뜨면 아래 "에러 팝업이 떴어요" 항목을 보세요. |
| **에러 팝업이 떴어요** | 팝업 안의 **[📋 로그 복사]** 버튼을 눌러 오류 내용을 클립보드에 복사할 수 있습니다. 복사된 내용은 `UA`·`URL`·오류 요약·최근 콘솔 로그 50줄을 담고 있으며, **운영자에게 붙여넣어 문의**하면 원인 파악에 큰 도움이 됩니다. |
| 로컬로 올린 PMX/PMD 모델이 **회색/흰색 단색**으로 보임 | PMX·PMD는 텍스처(눈·피부 이미지)가 원래 모델 파일과 *같은 폴더* 에 별개 파일로 있어야 정상 표시됩니다. 글쓰기 페이지의 "텍스처 번들" 섹션에서 **폴더 업로드** 또는 **ZIP 업로드** 로 텍스처를 함께 올리면 정상 색으로 보입니다. 번들을 안 올리면 회색으로 표시되는 것이 의도된 동작. |
| 모션 버튼을 눌러도 꿈쩍도 안 함 | 그 모델에 해당 "뼈대"가 없는 경우입니다. 상태 표시에 "본을 찾을 수 없음"이 뜨면 정상. 다른 모션을 시도하거나 VMD 파일을 드롭해 보세요. |
| VMD를 드롭했는데 "본 매핑 불일치" | 해당 VMD가 이 모델과 호환되지 않는 뼈대 이름을 씁니다. 다른 VMD로 시도. |
| 모바일에서 발열·느려짐 | 3D 렌더링은 휴대폰에 부담이 됩니다. PC에서 여는 게 쾌적합니다. |
| 뷰어 화면이 새카맣고 아무것도 안 나옴 | 그래픽 드라이버가 WebGL을 막은 경우. 브라우저 주소창에 `chrome://gpu` (엣지는 `edge://gpu`) 입력해 *WebGL: Hardware accelerated* 가 떠 있는지 확인. 아니라면 드라이버 업데이트 필요. |
| 실수로 "PR로 공유하기"를 눌러버렸어요 | 걱정 마세요 — PAT(토큰)를 설정한 적이 없으면 "PAT가 없습니다"라는 안내만 뜨고 아무 일도 일어나지 않습니다. |

## 7. "내 정보가 어디로 보내지나요?" — 프라이버시 요약

| 당신이 하는 행동 | 데이터가 어디로 가나 |
| --- | --- |
| 사이트 접속 | GitHub Pages 서버에서 HTML/JS/CSS를 받아옵니다. 브라우저가 표준적으로 하는 일과 동일. |
| 목록 페이지를 봄 | 같은 GitHub Pages에서 `posts.json`과 썸네일 이미지를 읽어옵니다. |
| 모델을 클릭해 뷰어로 감 | 공개 게시물이면 GitHub Pages에서 모델 파일을 받아옵니다. 로컬 게시물이면 당신 기기 안에서만 처리. |
| 외부 URL 배경이 있는 게시물을 봄 | 작성자가 지정한 그 외부 주소로 한 번 이미지 요청이 나갑니다. (CDN, 드롭박스 등일 수 있음) |
| 글쓰기 페이지에서 로컬 저장 | **아무 곳에도 나가지 않습니다.** 전부 당신 브라우저 내부(IndexedDB)에 저장. |
| VMD/BVH/모델 파일 드롭 | **아무 곳에도 나가지 않습니다.** 브라우저 내부에서만 처리. |

회원가입이 없으므로 이메일·비밀번호 같은 개인정보는 이 사이트에 없습니다.

## 9. 용어 사전

| 용어 | 쉬운 설명 |
| --- | --- |
| **브라우저** | 웹사이트를 여는 프로그램. 크롬, 엣지, 파이어폭스, 사파리 등. |
| **URL / 주소** | `https://...`로 시작하는 웹사이트 주소. |
| **3D 모델** | 3차원 캐릭터 데이터 파일. 이 사이트가 다루는 건 `.vrm`, `.pmx`, `.pmd`. |
| **VRM** | 아바타용 3D 포맷. 브이튜버·VR 쪽에서 많이 쓰임. |
| **PMX / PMD** | MMD(MikuMikuDance)용 3D 포맷. 미쿠 같은 캐릭터가 이 형식인 경우가 많음. |
| **VMD / BVH** | 모션(움직임) 파일. 모델은 그대로 두고 춤 등을 갈아끼우는 파일. |
| **드래그&드롭** | 마우스 왼쪽 버튼을 누른 채로 파일을 끌어다 다른 창 위에서 떨어뜨리는 동작. |
| **썸네일** | 목록에서 보이는 작은 미리보기 이미지. |
| **IndexedDB** | 브라우저 내부에 큰 데이터를 저장할 수 있는 공간. 쿠키보다 훨씬 넓음. |
| **WebGL** | 브라우저가 3D 그래픽을 그리게 해주는 기술. 모든 최신 브라우저에 기본 포함. |
| **PR** | Pull Request의 줄임말. GitHub에 글(파일)을 공유할 때 쓰는 "이 변경 반영해 주세요"라는 요청. *구경만 할 거면 몰라도 됩니다.* |
| **PAT** | Personal Access Token. GitHub에 글을 쓸 때 쓰는 비밀번호 대체용 열쇠. *구경만 할 거면 몰라도 됩니다.* |
